## 📖 Sobre el proyecto

Portfolio de una sola página con estética **futurista industrial** que fusiona dos mundos: el de la sala de máquinas (compresores, refrigeración, NH₃) y el del desarrollo web (código, circuitos, datos). Pensado como carta de presentación para roles que combinan perfil técnico industrial + capacidades de desarrollo.

## ✨ Características

- **Hero 3D interactivo** con Three.js: compresor frigorífico abstracto del que emergen fragmentos de código
- **Modo dual**: paleta cian/azul (Modo Dev) vs ámbar/naranja (Modo Técnico) con cambio en caliente
- **Timeline de experiencia** con ilustraciones SVG animadas únicas por empresa
- **Panel de control de habilidades** con medidores tipo industrial y módulos tipo circuito integrado
- **Terminal interactiva real** con comandos (`help`, `whoami`, `skills`, `github`, `matrix`, etc.) e historial
- **Audio sintetizado** vía Web Audio API (sin archivos externos)
- **Easter eggs** ocultos (incluido Konami Code)
- **100% responsive** (móvil, tablet, desktop)
- **Sin dependencias pesadas**: vanilla JS + 1 CDN (Three.js)

## 🚀 Cómo correrlo localmente

No requiere build ni servidor. Solo:

```bash
git clone https://github.com/TU_USUARIO_GITHUB/sergio-martin-rodriguez.github.io.git
cd sergio-martin-rodriguez.github.io
# Abrir index.html en cualquier navegador moderno
```

O directamente:

```bash
# Doble clic sobre index.html
```

> Nota: para que cargue Three.js desde la CDN, se necesita conexión a internet la primera vez.
